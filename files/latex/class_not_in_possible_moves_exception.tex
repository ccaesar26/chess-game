\doxysubsection{Detailed Description}
Exception class for moves that are not within the list of possible moves. 

This exception class is derived from the {\ttfamily \doxylink{class_invalid_move_exception}{Invalid\+Move\+Exception}} class and is used to represent errors related to moves that are not included in the list of possible moves for a chess piece. It provides constructors for creating exceptions with custom error messages.
