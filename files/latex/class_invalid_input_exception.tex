\doxysubsection{Detailed Description}
Exception class for invalid input errors. 

This exception class is derived from the {\ttfamily \doxylink{class_chess_exception}{Chess\+Exception}} base class and is used to represent errors related to invalid input in the chess game. It provides constructors for creating exceptions with custom error messages.
