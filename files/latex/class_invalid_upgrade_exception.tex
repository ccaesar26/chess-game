\doxysubsection{Detailed Description}
Exception class for invalid upgrades during pawn promotion. 

This exception class is derived from the {\ttfamily \doxylink{class_invalid_input_exception}{Invalid\+Input\+Exception}} class and is used to represent errors related to invalid upgrades during pawn promotion in the chess game. It provides constructors for creating exceptions with custom error messages.
