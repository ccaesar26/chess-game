\doxysubsection{Detailed Description}
Exception class for moves that are out of bounds on the chessboard. 

This exception class is derived from the {\ttfamily \doxylink{class_invalid_move_exception}{Invalid\+Move\+Exception}} class and is used to represent errors related to moves that are outside the bounds of the chessboard. It provides constructors for creating exceptions with custom error messages.
