\doxysubsection{Detailed Description}
Base exception class for chess-\/related errors. 

This exception class is derived from {\ttfamily std\+::logic\+\_\+error} and serves as the base class for all chess-\/related exception classes. It provides constructors for creating exceptions with or without custom error messages.
