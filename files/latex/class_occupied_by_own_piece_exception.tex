\doxysubsection{Detailed Description}
Exception class for moves to squares occupied by own pieces on the chessboard. 

This exception class is derived from the {\ttfamily \doxylink{class_occupied_square_exception}{Occupied\+Square\+Exception}} class and is used to represent errors related to moves to squares that are already occupied by own pieces on the chessboard. It provides constructors for creating exceptions with custom error messages.
