\doxysubsection{Detailed Description}
Exception class for moves from empty initial squares on the chessboard. 

This exception class is derived from the {\ttfamily \doxylink{class_invalid_move_exception}{Invalid\+Move\+Exception}} class and is used to represent errors related to moves from empty initial squares on the chessboard. It provides constructors for creating exceptions with custom error messages.
